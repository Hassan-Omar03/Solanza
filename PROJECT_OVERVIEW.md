# TechVision Solutions - Premium Software House Website

A beautiful, fully-featured software house website built with Next.js, featuring smooth animations, comprehensive services showcase, and payment integration.

## Features

### Pages
- **Home** - Stunning hero section with stats, features, services preview, and testimonials
- **About Us** - Company story, mission, vision, values, team members, and achievements
- **Services** - 30+ services with filtering, detailed descriptions, and pricing
- **Portfolio** - Project showcase with case studies, results, and filtering
- **Contact** - Contact form, business info, pricing packages, and Google Maps
- **Payment Demo** - Secure payment form demonstration

### Key Features
- Beautiful gradient designs with blue/cyan color scheme
- Smooth Framer Motion animations throughout
- Fully responsive design (mobile, tablet, desktop)
- High-quality free images from Pexels
- Professional UI components from shadcn/ui
- Payment integration ready (Stripe compatible)
- SEO-optimized structure
- Fast performance and modern UX

## Getting Started

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
\`\`\`
